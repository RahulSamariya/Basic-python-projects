# 🧮 Python GUI Calculator

A sleek, functional, and visually polished calculator built with **Python** and **Tkinter**.

---

## ✨ Features at a Glance

| Category | Functions |
| :--- | :--- |
| **🔢 Basic** | `+`, `-`, `x`, `÷` |
| **🧪 Scientific** | `√` (Square Root), `^` (Power), `!` (Factorial), `π` (Pi) |
| **⚙️ Advanced** | `%` (Modulo), `//` (Floor Division), `+/-` (Sign Toggle), `∑` (Summation) |
| **🎨 UI/UX** | Custom Palette, Centered Window, Non-resizable Layout |

---

## 🎨 Visual Palette

The calculator uses a modern, "Stone & Blush" theme:
- 🔘 **Buttons:** Soft Stone Gray (`#EAE6E3`)
- 🎀 **Special Keys:** Warm Blush Pink (`#ECCDC4`)
- 🖥️ **Display:** Lake Blue (`#1F4045`) with Soft Stone text

---

## 🚀 Getting Started

### 📋 Prerequisites
*   **Python 3.x** installed on your system.

### 🏃 How to Run
```bash
# 1. Open your terminal
# 2. Navigate to the project folder
# 3. Run the app:
python calculator.py
```

---

## 📂 Project Structure

*   📄 `calculator.py` — **The Heart:** GUI logic, custom math mapping, and the main loop.
*   📄 `demo.py` — **Toolbox:** A helper script for testing keyboard event bindings.
